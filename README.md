# rn-neti-mobile-image-viewing

React Native library for image viewing

## Installation

```sh
npm install rn-neti-mobile-image-viewing
```

## Usage


```js
import { multiply } from 'rn-neti-mobile-image-viewing';

// ...

const result = await multiply(3, 7);
```
